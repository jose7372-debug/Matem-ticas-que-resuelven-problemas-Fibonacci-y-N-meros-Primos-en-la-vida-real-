## 📝 Descripción del Problema Real
Ahorrar dinero de forma constante suele ser un reto. Este proyecto propone un método de **ahorro progresivo basado en la serie de Fibonacci**. En lugar de guardar una cantidad fija, el usuario incrementa su depósito mensual siguiendo este patrón natural de crecimiento. Esto permite empezar con montos muy bajos y aumentar el hábito del ahorro de manera escalonada y orgánica.

## 🧮 Explicación del Algoritmo
La serie de Fibonacci comienza con los números `0` y `1`, y cada término siguiente es la suma de los dos anteriores (`0, 1, 1, 2, 3, 5, 8, 13...`). 

El programa utiliza un ciclo lógico (`for`) en JavaScript que calcula dinámicamente el valor correspondiente a cada mes sin necesidad de usar vectores o arrays complejos, acumulando el total general directamente mediante variables simples:
* `a` (término anterior)
* `b` (término actual)
* `c` (suma de ambos para el siguiente mes)
