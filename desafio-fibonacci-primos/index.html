Contexto del Problema Real

Ahorrar dinero de forma constante suele ser difícil. Para solucionar esto, aplicamos un método de ahorro progresivo basado en la secuencia matemática de Fibonacci. En lugar de guardar una cantidad fija, el usuario aumenta su ahorro mes a mes siguiendo el patrón natural de crecimiento, permitiendo acumular una buena suma de dinero de manera escalonada.

¿Cómo funciona el algoritmo?

La serie de Fibonacci comienza con 0 y 1. Cada número siguiente es la suma de los dos anteriores (0, 1, 1, 2, 3, 5, 8...). El programa utiliza un ciclo para calcular el ahorro de cada mes sumando las variables de manera consecutiva, acumulando el total general sin necesidad de usar estructuras de almacenamiento complejas.
